# university-project-reports
This repository collects the reports and analyses for key projects developed during my Master's studies in Information Technology Engineering at the Università degli Studi di Brescia, highlighting my interests and practical experience.

---

## 1. Master Thesis: An Experimental Approach to Evaluating the Use of Advanced AI Tools in Radiology Task Learning Processes

| Topic | Summary |
| :--- | :--- |
| **Human-AI Collaboration in Medical Training** | Experimental analysis evaluating the impact of AI on the clinical training of 53 radiology residents. The study assesses the effectiveness of an neural architecture in learning a radiology score through different learning modalities, demonstrating that a human-first protocol is the most effective approach to accelerate clinical skill acquisition. |

---

## 2. Image Data Analysis Course: Deep Convolutional Neural Networks for Environmental Monitoring

| Topic | Summary |
| :--- | :--- |
| **Deep Convolutional Neural Networks** | Implementation and practical application of different CNN models for image classification. The project involved leveraging advanced architectures like VGG and ResNet to classify images capturing water turbidity and vortices in the Iseo lake. |

---

## 3. Artificial Intelligence Course: Multi-Agent Reinforcement Learning with PPO

| Topic | Summary |
| :--- | :--- |
| **Multi-Agent Reinforcement Learning** | In-depth analysis of state-of-the-art research in AI, using OpenAI works like Hide-and-Seek and OpenAI Five on Dota2. The report includes a detailed study of Multi-Agent Reinforcement Learning and algorithms like Proximal Policy Optimization (PPO) and how they have been applied to such works. |

---

## 4. Optimisation Algorithms Course: the Clustered Multidimensional Knapsack Problem (CMKP)

| Topic | Summary |
| :--- | :--- |
| **Combinatorial Optimisation** | Development of a solution for the Clustered Multidimensional Knapsack Problem (CMDKP), a combinatorial optimization task. The project focused on implementing, testing, and improve the Kernel Search heuristic, in order to produce good feasible solutions in a limited-time environment. |

---

## 5. Robotics Course: Action Model Learning with Guarantees

| Topic | Goal |
| :--- | :--- |
| **Action Model Learning** | Analysis and Implementation of the "Action Model Learning with Guarantees" approach by E. Scala and D. Aineto. The project aims to develop an algorithm implementation that learns sound and complete PDDL action models from observed agent transitions, applying the methodology to standard literature problems. |

[Code repository](https://github.com/claudiometelli/action-model-learning-guarantees)
